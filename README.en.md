# 二手交易管理系统SSM3

#### Description
二手交易管理系统3拥有两个角色，分别为管理员和用户，具体功能如下：
管理员：物品类型管理、物品管理、订单管理、用户管理
用户：浏览物品，购买物品，搜索物品，发布物品，查看、修改个人信息，评论物品等
